# Apollonia Events

A premium event reservation platform — a modern, calm, handcrafted experience inspired by
Ancient Greek aesthetics (marble, Aegean blue, antique gold, ivory, olive), without the clichés.

## Features (planned)

- Public marketing site: venue, events, and gallery
- Reservation requests (date, time, event type, guest count, contact, notes)
- Admin authentication
- Admin reservation calendar with status management and manual entries
- Admin dashboard
- Email reminders for upcoming reservations (later phase)

## Stack

- Next.js (App Router) + TypeScript
- Tailwind CSS + shadcn/ui
- PostgreSQL
- Server-side logic via Server Actions / Route Handlers
- Auth for admin

## Status

Early scaffolding. Built step by step — see milestones in the project plan.
